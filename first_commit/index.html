Development Guide

First Commit

Do not make any changes from the FTP file.

  1. Navigate to Source Control.
  2. Stage all file changes by clicking the + button on the right side of Changes.
  3. Type "first commit" in the message field.
  4. Click the Commit button.
  5. Click the three dots on the upper right side of Source Control and select Push.
  6. Click the three dots again and select Pull.

Create Local Branch

  1. At the bottom left of the VS Code window, click Branch.
  2. Click Create New Branch.
  3. Type "local" and press Enter.

Folder Structure

  1. Rename the folder imagemaps to imagemap.
  2. Move the folders imagemap and graphics to www.
